# 🌱 Plant Disease Detection using CNN

An AI-powered Plant Disease Detection System developed using Convolutional Neural Networks (CNN) and TensorFlow. The model predicts plant diseases from leaf images, helping in early disease identification and improving crop health.

---

## 🚀 Features

- Detects plant diseases from leaf images
- CNN-based Deep Learning model
- Trained on the PlantVillage Dataset
- Supports 38 different plant disease classes
- Image preprocessing and prediction
- High prediction accuracy (~95%)

---

## 🛠️ Tech Stack

- Python
- TensorFlow
- Keras
- OpenCV
- NumPy
- Google Colab
- Jupyter Notebook

---

## 📂 Dataset

PlantVillage Dataset (38 classes)

- Model: CNN
- Validation Accuracy: ~95%
