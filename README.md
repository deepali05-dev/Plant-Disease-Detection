# 🌱 Plant Disease Detection using Deep Learning

An AI-powered Plant Disease Detection System developed using **Convolutional Neural Networks (CNN)** to identify plant diseases from leaf images. The project was built to explore how Deep Learning can help in the early detection of crop diseases.

---

## 📌 Project Overview

This project classifies plant leaf images into different disease categories using a CNN model trained on the PlantVillage dataset.

To evaluate different machine learning approaches, **CNN performance was compared with ANN and SVM models**, with CNN achieving the best overall performance.

---

## ✨ Features

- 🌿 Plant disease classification from leaf images
- 🧠 CNN-based Deep Learning model
- 📊 Performance comparison with ANN and SVM
- 🖼️ Image preprocessing before prediction
- 📈 High validation accuracy (~95%)
- 📚 Trained on the PlantVillage dataset
